# Grambo Runs Static Site

This is a simple static HTML/CSS/JS website that can be opened locally or hosted on GitHub Pages.

## Test locally with VS Code + Live Server
1. Unzip the folder.
2. Open the `grambo-runs-site` folder in VS Code.
3. Install the Live Server extension if needed.
4. Right-click `index.html` and choose **Open with Live Server**.

## Deploy to GitHub Pages
1. Create a GitHub repository.
2. Upload these files to the repository root.
3. In GitHub: Settings → Pages.
4. Choose Deploy from branch → main → root.
5. Save.
